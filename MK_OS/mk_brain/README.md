# MK Brain

Higher-level cognitive modules: learning, memory, personality, reasoning, and embeddings.

## Subdirectories

- **learning/** - Learning engine for acquiring new knowledge from conversations
- **memory/** - Long-term and working memory management
- **personality/** - Response style, casual responses, and personality traits
- **reasoning/** - Reasoning engine for multi-step inference
- **embeddings/** - Word/sentence embeddings for semantic similarity
- **vector_search/** - Approximate nearest neighbor search over embeddings
- **fact_extractor/** - Extracts structured facts from unstructured text
- **cashe/** - Intelligent caching for frequent queries
- **daily_briefing.cpp** - Daily summary and proactive notifications
