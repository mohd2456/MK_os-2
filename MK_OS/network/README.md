# Network

Internet connectivity and real-time data access layer.

## Key Components

- **main_network.cpp** - Core networking with HTTP client (libcurl)
- **realtime_apis.cpp** - Live weather, time, and news API integration
- **search_engine.cpp** - DuckDuckGo and Wikipedia search with citation
- **autonomous_agent.cpp** - Self-directed web research agent
- **mk_network.cpp** - Network utilities and connection management
- **http.cpp** - Low-level HTTP request/response handling
- **parser.cpp** - HTML/JSON response parsing
- **update.cpp** - Update check and download mechanisms
