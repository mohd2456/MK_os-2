# Tools

Standalone utilities, code execution, file handling, and knowledge management.

## Key Components

- **code_runner.cpp** - Sandboxed execution of Python, Bash, and C++ code with timeout protection
- **file_reader.cpp** - Read text/code/PDF files with metadata extraction
- **image_analyzer.cpp** - PNG/JPEG header parsing, format detection, and dimension extraction
- **knowledge_ingest.cpp** - Knowledge ingestion pipeline for .mk format
- **knowledge_sync.cpp** - Sync knowledge across instances
- **llm_eater.cpp** - Convert LLM output into structured knowledge
- **telegram_bridge.cpp** - Telegram message routing bridge
- **generate_knowledge.py** - Python script to bulk-generate .mk knowledge files
- **knowledge_data/** - Raw knowledge data sources
