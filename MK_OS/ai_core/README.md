# AI Core

The central intelligence layer of MK OS. Contains all reasoning, generation, and language understanding modules.

## Key Components

- **Prometheus Engine** (`prometheus/`) - Master generation engine using fluid intelligence with droplets, birth chambers, and code fragments
- **Crystal Network** (`cxn/`) - Response generation via crystal resonance and semantic triggering
- **Consciousness Engine** (`mce/`) - Multi-layer thought assembly: atoms, chain reactions, word weaving, and echo memory
- **HRE** (`hre/`) - Hybrid Reasoning Engine with deep reasoning, pattern graphs, knowledge files, and compositional logic
- **Smart Router** - Classifies queries and dispatches to the best handler
- **Conversation Mode** - 602 conversation templates for natural dialog
- **Idea Engine** - Creative concept bridging by connecting unrelated domains
- **Math Solver** - Quadratic, calculus, unit conversion, and physics formulas
- **Neural Net** - Lightweight neural network for classification tasks
- **Self Improver** - Nightly knowledge growth and rule generation
